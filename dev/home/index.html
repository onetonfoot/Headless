A package for controling chrome through the devtools protocol.

Installation

Since the package is currently unregistered

import Pkg
Pkg.add("https://github.com/onetonfoot/Headless.git")

Ensure you already have chrome installed.

Getting Started

To open a instance of chrome with a single tab run.

julia> chrome = Headless.Chrome(headless=true, port=9522)
Chrome:
- running - true
- port - 9522
- num tabs - 1

Chrome should start with one tab, you can get the tab name with tabnames

julia> tabnames(chrome)
1-element Array{Symbol,1}:
 :tab1

Tabs can be used execute commands such as navigating to a new page.

julia> using Headless.Protocol: Page

julia> cmd = Page.navigate("https://www.google.com")
Headless.Protocol.Command(27989, "Page.navigate", Dict{String,Union{Nothing, String}}("referrer"=>nothing,"frameId"=>nothing,"transitionType"=>nothing,"url"=>"https://www.google.com"), nothing, nothing)

julia> chrome[:tab1](cmd)
ERROR: UndefVarError: isnothing not defined

Finally to close the browser

julia> close(chrome)
true
